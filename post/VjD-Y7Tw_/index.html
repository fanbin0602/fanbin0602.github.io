🔗W3C 的「弹幕规范」工作草案

W3C 是指万维网联盟，创建于1994年，是Web技术领域最具权威和影响力的国际中立性技术标准机构。平时我们知道的 Web 标准都是 W3C 制定的，这也就是为什么世界上大部分网页与大部分浏览器都是兼容的。近期 W3C 的技术标准文档里出现了一篇「弹幕规范」。W3C 的技术标准从开始到成为业界公认标准总共分为四个阶段，分别是：

  • 工作草案（WD）
  • 候选推荐标准（CR）
  • 提案推荐标准（PR）
  • W3C 推荐标准（REC）

目前，这份弹幕规范处于最初的工作草案阶段。

🔗 吴军的采访视频

前几天，吴军的一段采访视频在互联网上广泛传播。大家的讨论热点集中在了他对大型互联网公司的评价上——「谷歌平庸，苹果受困，百度落伍，腾讯没有 To B 的基因」。

但我看完视频发现，这些并不是他聊天的重点。他对 5G 和 IoT 的观点才是这段采访视频的主要内容。如果没时间看视频，我把他的观点总结一下：

  1. 在第一代互联网到第二代互联网转变的时候，有一些公司死掉了，有一些公司落伍了，腾讯和阿里幸存了下来，还有大量的新公司诞生了。到第三代互联网诞生的过程中，也会发生同样的事情。
  2. 第一代互联网到第二代互联网指的是传统互联网和移动互联网，也可以指机器互联和人互联，第三代互联网指的是万物互联，也就是 IoT。
  3. 在中国，还没有公司能确定在第三代互联网时代做大。格力卖出了大量的家电，但是没有往智能化的方向转变；米家的设备做的很好，但是服务方面还有路要走。在国外，Google Home 和 Amazon Alex 已经做得不错，比如 Alex 已经与大众汽车合作，做语音交互。
  4. 在前两代互联网的转变中，芯片由 Intel 转变到了 ARM，操作系统由 Windows 转变到了 Android，这是因为后者在信息处理效率上出现了数量级上的提升，这个规律在下一代互联网出现的过程中，同样适用。
  5. 科学技术是两件事，科学是把钱变成知识，技术是把知识变成钱，中国更擅长后者。

我比较认同他的观点。有一句话说「历史从不重复，但总是押韵」。我们可以从过去互联网行业的发展过程中，预测未来的发展。5G 的到来会为 IoT 提供比现在好得多的基础设施条件，预计十年内，IoT 会在中国发展地极度成熟。

🔗 QMUI Team 发布免费的 iOS UI 调试工具

腾讯的 QMUI 团队，最近发布了他们在内部使用的 iOS UI 调试工具 Lookin，开源免费。与著名的 macOS 平台的界面调试工具 Reveal 类似。一下是它的官方介绍：

Lookin 可以查看与修改 iOS App 里的 UI 对象，类似于 Xcode 自带的 UI Inspector 工具，或另一款叫做 Reveal 的软件。
但借助于“控制台”和“方法监听”功能，Lookin 还可以进行 UI 之外的调试。
此外，虽然 Lookin 主体是一款 macOS 程序，它亦可嵌入你的 iOS App 而单独运行在 iPhone 或 iPad 上。
最后，Lookin 完全免费。

由于是非营利性组织，所以该软件完全免费，不含任何内购或广告，亦无需注册任何账户

🔗 要在《乐队的夏天》翻唱一首歌，怎么处理版权才能不侵权？
